# 🍕 HJS Pizzaria - Sistema de Gestão e Encomendas

Sistema Full-Stack desenvolvido como projeto acadêmico para o curso de **Engenharia de Software (Unicesumar)**. O objetivo é gerenciar o fluxo completo de uma pizzaria, desde o cardápio dinâmico até o controle de produção na cozinha.

## 🚀 Tecnologias Utilizadas
* **Backend:** Java (Spring Boot / Maven)
* **Banco de Dados:** MySQL (Hospedado no Railway)
* **IDE:** Visual Studio Code
* **Versionamento:** Git & GitHub

## 📋 Funcionalidades Planejadas

### 👤 Painel do Cliente
* Visualização de cardápio dinâmico com fotos.
* Sistema de login e cadastro.
* Realização de pedidos com definição de forma de pagamento.
* Acompanhamento de progresso do pedido em tempo real.
* Avaliação de pedidos concluídos.

### 👨‍🍳 Painel da Cozinha
* Fila de pedidos por ordem de prioridade.
* Visualização de itens e observações (ex: "Sem cebola").
* Alteração de status: *Recebido* ➔ *Sendo Preparado* ➔ *Pronto*.

### 📊 Painel Administrativo
* Gestão completa de produtos (CRUD).
* Relatórios de lucros com filtros por dia, semana e mês.
* Monitoramento geral dos pedidos ativos.

## 🏗️ Estrutura do Banco de Dados
O banco está estruturado para suportar múltiplos tipos de usuários e relações complexas entre itens e pedidos, garantindo a integridade dos dados e agilidade nas consultas.
